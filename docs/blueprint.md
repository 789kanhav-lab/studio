# **App Name**: ChronoTrack

## Core Features:

- Start/Stop Timer: Initiate and halt the timer with precise timing.
- Record Lap Times: Capture and store individual lap times during a timing session.
- Reset Timer: Clear the current timer and lap time data to begin a new session.
- Display Elapsed Time: Show the total time elapsed since the timer was started.
- Lap Time Details: Present a detailed list of all recorded lap times, with the option to show timestamp of each lap.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to highlight the interactive elements against the dark backdrop, conveying precision.
- Background color: Dark gray (#222222) to provide contrast and a modern feel.
- Accent color: Light blue (#A9F3FF) as a secondary highlight for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, known for its modern, machined and neutral feel.
- Minimalist icons for start, stop, lap, and reset functions to enhance usability.
- Clean, uncluttered layout with key metrics prominently displayed.
- Subtle transition animations for timer updates and lap recording.
